SmartDirectives

File Inputs

{{fileName}}
  • The "file-input" attribute signifies the input element.
  • "file-show" attribute must equal the scope variable that represents the file title.
  • The "file-remove" attribute signifies the remove button.
      <div class="file-button-container">
        <span class='file-button btn btn-default btn-file'>
          <input file-input file-show="fileName" id="new_file" type="file" name="file">
          <span class="glyphicon glyphicon-paperclip"></span>
        </span>
        <span class="file-button-title"> { {fileName} } </span>
        <span file-remove file-id="new_file" file-show="fileName" ng-show="fileName" class="glyphicon glyphicon-remove"></span>
      </div>
