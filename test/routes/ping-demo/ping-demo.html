SmartPing

SmartPing is a way to quickly create and manage (many) looping timeouts.

Demo

Start ping 1 Stop ping 1 {{counters.1}}


Start ping 2 Stop ping 2 {{counters.2}}

Start a timeout loop be clicking on the Start button. This will increment the display value by one every second.

Cancel a timeout loop by clicking on the Stop button.

Notice how the timeout loops can be managed independently.
